1. ฟังก์ชันคืออะไร?

ฟังก์ชัน (Function) ใน Python คือกลุ่มของคำสั่งหรือโค้ดที่รวมตัวกันเพื่อทำงานเฉพาะเจาะจง การเขียนฟังก์ชันช่วยให้โค้ดมีความเป็นระเบียบ อ่านง่าย และสามารถนำกลับมาใช้ใหม่ได้ นอกจากนี้ ยังช่วยลดความซ้ำซ้อนในโค้ดและทำให้การดีบักหรือแก้ไขโค้ดทำได้ง่ายขึ้น ฟังก์ชันใน Python มีสองรูปแบบหลัก: ฟังก์ชันที่สร้างโดยผู้ใช้ - ผู้พัฒนาสามารถกำหนดฟังก์ชันเองได้เพื่อให้เหมาะสมกับความต้องการของโปรแกรม ฟังก์ชันที่มีอยู่แล้วใน Python - เช่น print(), len(), range() เป็นต้น ประโยชน์ของฟังก์ชัน: การใช้ซ้ำ (Reuse): สามารถเรียกใช้งานได้หลายครั้ง โดยไม่ต้องเขียนโค้ดเดิมซ้ำ โครงสร้างชัดเจน: ช่วยจัดระเบียบโค้ดให้ดูง่ายขึ้น ลดความซับซ้อน: ช่วยแยกส่วนการทำงานในโปรแกรมให้เป็นโมดูลเล็กๆ เพิ่มความยืดหยุ่น: สามารถรับค่าที่แตกต่างกันและคืนค่าที่ต่างกันได้ ตัวอย่างในชีวิตจริง: การกดปุ่ม "ปริ้นเอกสาร" บนคอมพิวเตอร์ คือการเรียกใช้ฟังก์ชันที่ทำหน้าที่ปริ้นเอกสาร การใช้แอปพลิเคชันสั่งอาหาร ฟังก์ชันจะถูกเรียกเพื่อคำนวณราคาอาหาร รวมภาษีและค่าจัดส่ง ใน Python การสร้างและใช้งานฟังก์ชันเป็นพื้นฐานที่สำคัญสำหรับการเขียนโปรแกรมทั้งขนาดเล็กและใหญ่
